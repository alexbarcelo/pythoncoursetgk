Comprehensions

List comprehensions


squares = [x**2 for x in range(10)]
# squares ->  [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
powers_of_two = [2**x for x in range(10)]
# powers_of_two -> [1, 2, 4, 8, 16, 32, 64, 128, 256, 512, 1024]
    

fancy_filter = [x for x in squares if x in powers_of_two]
# squares ->  [4, 16, 64]

Function args and kwargs


def af(a, b, c, d=4, e="5", f=None, g={}):
    g[d] = a
    if f is not None:  # not exactly the same as `if f:`
        g[e] = f
    return g

Correct statements:


af(1, 2, 3)  # <-- returns {4: 1}
af(1, 2, 3, 4, 5)
af(1, 2, 3, f=4, g={})
af(1, 2, 3, f=5, e="4")
af(a=1, b=2, c=3, d=4, e=5)

Also correct:


args = (1, 2, 3)
kwargs = {d: 4, e: 5, f: 6, g: {}}
af(*args, **kwargs)
